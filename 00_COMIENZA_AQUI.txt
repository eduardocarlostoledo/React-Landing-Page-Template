   └─ Variables: SERVICE_ID, TEMPLATE_ID, PUBLIC_KEY

2. COLOCA EL PDF
   ├─ Copia sistema55.pdf a public/
   └─ El archivo debe existir para download

3. CREA LINK MERCADOPAGO
   ├─ Ve a MercadoPago
   ├─ Crea suscripción de $5 USD/mes
   ├─ Copia el link
   └─ Reemplaza en Sistema55.jsx línea 63

4. PRUEBA LOCALMENTE
   ├─ npm start
   ├─ Abre http://localhost:3000/sistema-5-5
   ├─ Completa formulario
   └─ Verifica ambas opciones (PDF + Pago)

5. DEPLOY A PRODUCCIÓN
   ├─ npm run build
   ├─ Deploy a Vercel/Netlify/Tu-hosting
   ├─ Configura .env en plataforma
   └─ Verifica en tu dominio
